# Claude Desktop Unofficial Binaries

MFs at Claude does not provide Linux binaries. So [this][01] dude provides
claude binaries for Linux, both for AMD64 and ARM64. In the [relases][02] page
there are binaries for Fedora based and debian based distros but not
for Arch Linux. So in [AUR][03] same dude shared package for x86_64. But
this electron app is also works on ARM64.

[01]: https://github.com/patrickjaja/
[02]: https://github.com/patrickjaja/claude-desktop-bin
[03]: https://aur.archlinux.org/packages/claude-desktop-bin
